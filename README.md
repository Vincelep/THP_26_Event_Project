# 26ème jour - The Event Project

## 💎 Qu'est-ce donc ?
Une app **Ruby On Rails** 

## ⚙ Installation
1. Clone ce repo (ou download-le)
2. Place-toi dans le dossier et va dans ton magnifique terminal 😍 :
3. `$ bundle install` 
4. `$ rails db:create`
5. `$ rails db:migrate` 
6. `$ rails db:seed`
7. `$ rails server`
